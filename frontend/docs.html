Overview

API Tool is a premium platform for individuals and organizations to test APIs, save collections, track request history, manage plans, and publish dynamic product pages.

How to send a request

Open the API Tester workspace, choose an HTTP method, enter the API URL, add headers and request body if needed, then click Send Request.

After sending, you will see the response body, full response, status code, and response time.

How collections work

Collections help you save useful API requests so you can reuse them later. You can load, edit, delete, export, and import saved collections.

How history works

Every request you send is stored in your history with request method, status code, URL, and date. This helps you review your testing activity.

Premium plans

Users can choose a premium plan from the purchase page. After confirming payment, the selected plan becomes active on the user dashboard.

Admin panel

The admin panel is separate from the user dashboard. Admin users can manage CMS pages, users, purchases, collections, and system-wide history.
